

In `Number King`, you manipulate number like a king!


Batch Number Manipulation
=========================

Useful symbols:

| Symbol | Description                               |
| ------ | ----------------------------------------- |
| `x`    | the selected number                       |
| `i`    | the index of the selection (start from 0) |

Supported functions / constants:

- `sin`
- `cos`
- `tan`
- `log`
- `e`
- `pi`
