
Batch Number Manipulation Reference
===================================

You can use Python expression to manipulate all the numbers you selected.

Some useful variables:

| Symbol | Description                               |
| ------ | ----------------------------------------- |
| `x`    | the selected number                       |
| `i`    | the index of the selection (start from 0) |

Supported functions / constants:

 - `ceil`
 - `copysign`
 - `fabs`
 - `factorial`
 - `floor`
 - `fmod`
 - `frexp`
 - `fsum`
 - `isfinite`
 - `isinf`
 - `isnan`
 - `ldexp`
 - `modf`
 - `trunc`
 - `exp`
 - `expm1`
 - `log`
 - `log1p`
 - `log2`
 - `log10`
 - `pow`
 - `pow`
 - `sqrt`
 - `acos`
 - `asin`
 - `atan`
 - `atan2`
 - `cos`
 - `hypot`
 - `sin`
 - `tan`
 - `degrees`
 - `radians`
 - `acosh`
 - `asinh`
 - `atanh`
 - `cosh`
 - `sinh`
 - `tanh`
 - `erf`
 - `erfc`
 - `gamma`
 - `lgamma`
 - `pi`
 - `e`
